Primeiro você precisa ter o Visual Studio Code, uma platafoma para fazer sites, depois você precisa adicionar uma extenção chamada "Live server", após isso crie uma pasta clicque em arquivo e abrir pasta, clique em novo arquivo e escreva index.html

A tela console é ultilizada para testar o site, ver o valor das variaveis. Para abrir a tela console aperte a tecla F12 e depois clique em console, para imprimir qualquer coisa no console no codigo em Html você precisa escrever a tag script e dentro dela colocar console.log() ex: console.log("Olá")
